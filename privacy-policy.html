ArnMesh Privacy Policy

Effective date: April 24, 2026

Overview

ArnMesh is a private mesh network tool for your own trusted devices. It is designed to let Windows PCs and Android phones reach each other through a controlled private mesh while keeping device keys, profiles, and network membership data local whenever possible.

Data Stored On Your Devices

ArnMesh may store network profiles and profile names, device names and platform labels, overlay IP addresses and mesh CIDR values, public keys and public key fingerprints, private tunnel keys needed by that device, pairing and approval state, direct endpoint candidates, last-seen timestamps, and local connection status.

On Windows this data is managed by ArnMesh Service. On Android it is stored inside the app private data folder.

Pairing And Coordinator Data

When a coordinator server is configured, ArnMesh may send the minimum metadata needed for pairing and device discovery: network id, device id, device name, platform, overlay IP, public key fingerprint, public tunnel key, direct UDP endpoint candidates, approval status, and last-seen timestamps.

The coordinator is not intended to receive device private keys.

Tunnel Traffic

ArnMesh tunnel payloads are encrypted between approved devices. The coordinator can help devices discover each other, but it should not need private keys or decrypted traffic.

In direct-only mode, packet traffic is sent directly between devices and not through the coordinator. If relay fallback is configured, the relay is intended to forward encrypted packets only.

Data ArnMesh Does Not Intentionally Collect

ArnMesh does not intentionally collect browsing history, file contents, messages, contacts, photos, precise location history, or the contents of RDP, SSH, game, browser, or app traffic.

Security Responsibility

ArnMesh is built to avoid publishing private data, but no app or network connection can guarantee absolute protection. You are responsible for keeping your devices, operating systems, passwords, recovery accounts, and coordinator credentials secure.

To the maximum extent allowed by law, Arn1ko Studio is not responsible for data loss, unauthorized access, or disclosure caused by compromised devices, exposed credentials, third-party services, unsafe networks, user configuration, malware, or use of experimental builds.

Security Notes

ArnMesh is still in active development. Early builds should be treated as experimental. Do not rely on test builds as the only protection for high-risk production systems until signed updates, audited storage encryption, stronger server hardening, rate limits, and production incident handling are complete.

Your Choices

You can remove devices, reset local profiles, uninstall the app, or delete coordinator-side network data when server management tools are available. Removing a device stops it from being trusted by that network profile.

Changes

This policy may change when ArnMesh adds production accounts, paid plans, updater services, hosted coordinator features, or relay options.
